Function Parameter

  • Kita bisa mengirim informasi ke function yang ingin kita panggil
  • Untuk melakukan hal tersebut, kita perlu menambahkan parameter atau argument di function yang sudah kita buat
  • Parameter ditempatkan didalam kurung() dideklarasikan didalam method.
  • Parameter bisa lebih dari satu, atau lebih dari satu, harus dipisahkan dengan tanda koma.
