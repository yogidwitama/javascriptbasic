For Loop

  • For adalah salah satu kata kunci yang bisa digunakan untuk melakukan perulangan
  • Blok code yang terdapat dalam for akan selalu diulangi selama kondisi for terpenuhi

Sintak For Loop

for(init statement; kondisi; post statement){ //block perulangan}

  • Init statement akan dieksekusi hanya sekali dari awal sebelum perulangan
  • Kondisi akan dilakukan pengecekan dalam setiap perulangan, jika true perulangan akan dilakukan, jika false perulangan akan berhenti
  • Post statement akan dieksekusi setiap kali diakhir perulangan
  • Init statement, Kondisi dan Post Statement tidak wajib diisi, jika kondisi tidak diisi, berarti kondisi selalu bernilai true
