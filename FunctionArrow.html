Arrow Function

  • Arrow Function adalah alternatif pembuatan function yang lebih sederhana dari function biasanya
  • Terdapat limitasi dan juga tidak bisa digunakan di semua situasi
  • Dinamakan Arrow function karena menggunakan tanda =>
  • Kekurangan arrow funtion: ❎tidak memiliki fitur argumen object ❎tidak bisa menggunakan function generator ❎tidak bisa menggunakan this ❎tidak bisa mengakses super

Arrow Function tanpa block

  • Jika sebuah arrow function isinya sederhana, misalnya satu baris
  • kita bisa membuatnya tanpa harus menggunakan block

Arrow Functiob Return Value

  • Arrow function bisa mengambalikan value sama seperti biasanya
  • Jika menggunakan block mka perlu menggunakan kata kunci return
  • Jika menggunakan block tidak perlu kata kunci return

Arrow Function tanpa kurung parameter

  • Jika parameter di arrow function hanya satu, kita bisa tidak menggunakan kurung pada parameter

Arrow Function sebagai parameter

  • Karena arrow function sama seperti anonymous function
  • jadi juga bisa menggunakan arrow function sebgai parameter di function lain
