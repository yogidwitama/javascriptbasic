Fucntion Generator

  • Fucntion Generator adalah function yang digunakan untuk membuat data genarator
  • Generator adalah data yang bisa di interaksikan dengan Array
  • Untuk membuat function generator, perlu menggunakan tanda * (bintang) setelah kata function
  • Untuk mengembalikan data di tiap iterasi, bisa gunakan kata kunci yield diikuti datanya

Function Generator Komleks

Lazy Evaluation

  • Generator itu sifatnya lazy
  • Jadi artinya ketika data belum diambil dari generator, maka yield selanjutnya tidak akan dieksekusi
